Примечания, ограничения калькул ятора:

Вводить данные можно в любое поле, перед которым синие названия. Захват данных - соответствующей кнопкой ввода нужных данных.

Вывод производится из всех полей данных, кроме тех, которые Вы использовали для ввода

Точность перевода при расчете координат обычно не хуже, чем ± 0.1"

Знак "-" для d и b в форме (o ' ", первая строка - для экваториальных данных, а в нижней форме - для Галактических координат)  вставляется нажатием специального поля перед полем o. Знак может быть также вставлен и в поле для o, но тогда в случае -1o < d < 0o в этом поле значение станет равным "0" и знак будет утерян.

Эклиптические координаты считаются на основе текущего угла наклона Эклиптики ("E/El obl") с вычислением нутации.

Галактические координаты считаются со следующими параметрами перевода: North Galactic Pole equatrial coordinates (1950) aN = 192o.25, dN = 27o.4, Galactic longitude of ascess node on equator l0 = 33o.0.
